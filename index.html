✖ Warning! You are using Internet Explorer. IE only supports the 'text' and 'URL' transfer types so your experience may be limited. Also IE9 and below have no support for dragging images from your desktop to the page

Any draggable elements (images, link tags, selected text) can be dropped here, but only dragged images from external sites / uploaded images will be displayed

Some images are wrapped around an anchor tag, these won't work as the API will consider the drop to be a 'link' and not an 'image'
